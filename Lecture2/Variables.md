before es6: we were using the var keyword to declare the variables
issues with var:
1. redeclartion allowed
2. reassignment allowed
3. no block scope

for example
var.js



"let" keyword:
1. redeclartion not allowed
2. reassignment allowed
3. block scope

for example
let.js

"const" keyword:
1. redeclartion not allowed
2. reassignment not allowed
3. block scope
4. must be initialized at the time of declaration

"console.log()" is used to print something on the console.
